 Installation
 ================
 1) Copy twitterblock into 'extension' folder
  
	
 2) Activate twitterblock
   In administrator user interface, click 'setup' tab->'extensions' menu, select 'twitterblock', click button 'Apply Changes'.
   or in settings/override/site.ini.append.php, add acivation configuration under "ExtensionSettings" section:
     [ExtensionSettings]
     ActiveExtensions[]=twitterblock

 3) Regenerate autoloads
    In administrator user interface, click 'setup' tab->'extensions' menu, select 'twitterblock', click button 'Regenerate autoload arrays for extensions',
    or in eZ Publish installation folder, run "php bin/php/ezpgenerateautoloads.php -e"
    
 4) Clear cache
    Clear INI and template caches. (from admin 'Setup' tab or commandline)
